# Changelog

All notable changes to the NimbyScript extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-12-11

### Added

- Initial release
- Syntax highlighting via TextMate grammar
- Semantic token support for enhanced highlighting
- Real-time diagnostics and error reporting
- Autocompletion for:
  - Keywords and language constructs
  - Primitive types (i8, i16, i32, i64, u8, u16, u32, u64, f32, f64, bool)
  - Game API types, methods, and fields
  - User-defined structs, enums, and functions
  - Callback functions with signature templates
- Hover information with documentation for:
  - API types, methods, and fields
  - User-defined symbols
  - Type annotations
- Signature help for function and method calls
- Document outline (symbols view)
- Support for NIMBY Rails v1 API definitions
